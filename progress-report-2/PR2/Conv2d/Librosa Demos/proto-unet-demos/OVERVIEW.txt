WHAT EACH DEMO DOES:
--------------------

Demo 1: Skip Connections
- Compares reconstruction WITH vs WITHOUT skip connections
- Shows information preservation vs. loss
- Output: 2 audio files + comparison visualization

Demo 2: Encoder Hierarchy
- Builds 4-level encoder pyramid
- Extracts multiple feature types (edges, blur, etc.)
- Shows receptive field growth
- Output: 3 visualization figures

Demo 3: Decoder Reconstruction
- Tests 4 upsampling strategies:
  * Nearest neighbor
  * Bilinear interpolation
  * Transposed convolution
  * Sub-pixel convolution
- Computes quality metrics (PSNR, SSIM)
- Output: 4 audio files + 2 visualization figures

Demo 4: BatchNorm Stabilization
- Simulates training WITH vs WITHOUT BatchNorm
- Shows gradient flow improvement
- Analyzes activation distributions
- Output: 3 visualization figures

USAGE:
------
1. Install dependencies:
   pip install -r requirements.txt

2. Run individual demos:
   python demo1_skip_connections.py
   python demo2_encoder_hierarchy.py
   python demo3_decoder_reconstruction.py
   python demo4_batchnorm_stability.py

3. Check outputs:
   - Audio files in: output/
   - Visualizations in: figures/
